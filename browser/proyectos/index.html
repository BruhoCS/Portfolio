Frontend de FitHub

Pulsando este titulo te dirigirá a la página web!

Descripción

FitHub es un proyecto Frontend basado en un gimnasio el cual su objetivo es aportar una herramienta con la cual los usuarios puedan planificar sus entrenamientos ya sea de forma manual o mediante un IA.

Debido a que el proyecto está diseñado para permitir la visualización de distintas funcionalidades asociadas a diversos roles de usuario, no se ha implementado un sistema de seguridad completo. Para facilitar la comparación de las funcionalidades de los roles, se proporcionan las siguientes credenciales de acceso: Nombre de usuario: Administrador, Contraseña: abc123.. Nombre de usuario: Usuario, Contraseña: abc123..
